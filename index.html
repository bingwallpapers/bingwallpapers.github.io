Bingwallpapers

Download and set bing's wallpaper of the day as a desktop and screen lock wallpaper.

Installation

Installation

Bingwallapers requires the following to run

  • Qt v5.7+
  • Systems that supports GSettings

    • Install

    $ sudo dnf copr enable julekgwa/Bingwallpapers
$ sudo dnf install bingwallpapers


    The application starts in a minimized window (in your system tray), open it from system tray to maximize it.


    Gnome 3.26

    Gnome 3.26 users need to install, extensions like TopIcons Plus and kStatus/Appindicator, to show system tray. Gnome 3.26 has removed the legacy tray.
